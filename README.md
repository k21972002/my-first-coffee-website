# Personal Portfolio

A clean, Apple-inspired personal portfolio built with React + Vite.

## Sections

- **Hero** — Introduction and call to action
- **About** — Bio and skills
- **Work** — Selected web projects
- **Contact** — Enquiry form and social links

## Getting Started

```bash
npm install
npm run dev
```

## Customization

Edit `src/data/site.js` to update your name, bio, projects, and contact details.
